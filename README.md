# 档案管理语言 DGY

DGY 是一门用于计算机教学的，以 Forth 为灵感的中文编程语言。“DGY”是“档案管理语言”的汉语拼音缩写，即“Dang an Guan li Yu yan”的缩写。

它有两个核心理念：一是强调操纵数据，编程要以数据为中心；二是强调参考文档，编程是离不开文档的。

（本项目正在开发中）

## 示例程序

```
九九乘法表打印 =
/* 令 九九乘法表打印 无结果 */
    乘法算式打印 =
    /* 令 乘数1 乘数2 打印 无结果 */
        设 打印 = @打印
        设 乘数1 = #1
        设 乘数2 = #2
        设 积 = #1
        设 乘号 = *\**
        设 等号 = *=*
        令 *整数* 格式的 乘数1 打印
           *字符* 格式的 乘号 打印
           *整数* 格式的 乘数2 打印
           *字符* 格式的 等号 打印
           乘数1 乘数2 相乘
        结果存 积
        令 *整数* 格式的 积 打印
        无结果
    = 乘法算式打印

    设 A = 1
    设 B = 2
    存 1 1 到 #A #B
    重复执行
    检测 #A > 9 条件
       令 #A #B 乘法算式打印
          #B 加一
       无结果
       如果 #B = 9 就
          令 #A 加一
          无结果
       否则结束
    直到 成立
= 九九乘法表打印

令 九九乘法表打印 无结果
```
